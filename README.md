# Voice Recognition
------------------------
## What is This?

 - Using google voice API
 - For Android
 - 추가적인 activity dialog 나 Voice Typing IME 전환없이 인식
 - 키보드 음성인식(STT) 추가 이전 TOY project
 - Factory 패턴 적용

## Prerequisites

- Android Manifest 권한 추가
 ```xml
    <uses-permission android:name="android.permission.RECORD_AUDIO" />
    <uses-permission android:name="android.permission.INTERNET" />
```
### Install List

 - minSdk ```21```

 - Kotlin
